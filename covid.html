COVID-19 Database


Welcome to the COVID-19 Database!

  • Record A Vaccination allows you to record the vaccinations a patient has taken. If the patient is not already in the database, this will also allow you to add them.

  • Find a Vaccination Location allows yout o find all the locations that are providing a specific type of vaccination along with the number of doses they have in stock.

  • Search Vaccination Status allows you to view the vaccinations a patient has received.

  • Show Employees Allows you to see the doctors and nurses working at a specified vaccination site.
